# Deloitte Data Analytics Job Simulation

This repository contains 2 Tableau dashboards created as part of the Deloitte Australia Data Analytics Job Simulation.

## Dashboards

### 1. Equality Dashboard
Shows gender representation and pay equality across different roles and departments.
**Key Insight**: Visualizes workforce diversity metrics.

### 2. Unhealthy Dashboard  
Analyzes health-related factors and their impact on employee productivity.
**Key Insight**: Identifies trends in employee wellbeing.

## Tools Used
- Tableau Public
- Data Analytics
- Data Visualization

Completed as part of Deloitte's Job Simulation on Forage.
